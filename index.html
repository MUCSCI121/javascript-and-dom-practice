for Loop

Loops through a block of code a number of times.

      for(let i = 0; i < 5; i++) {
        // Do something
      }

while Loop

Loops through a block of code as long as a specified condition is true. 

      while(hour < timeToWakeUp) {
        // Do something
      }

do/while Loop

This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true. 

      do {
        // Do something
      } while(hour < timeToWakeUp);

if/else if/else Conditional

Specify blocks of code to execute based on whether the condition is true. 

        if (day === "Saturday"){
        // do something for Saturday
        }
        else if (day === "Sunday"){
        // do something Sunday
        }
        else {
        // do something for all other days of the week
        }

switch Conditional

Used to perform different actions based on different conditions.

        switch(feeling){
          case "happy":
            // do something because you are happy
          break;

          case "sad":
            // do something because you are sad
          break;

          default:
           // do something for all other cases
          break;
        }
